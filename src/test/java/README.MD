Тесты даны для демонстрации выриантов тестирования и представлены в ограниченном количестве.

Тестирование производится на базе h2 в памяти.

Варианты тестов:
- Тестирование создания json
- Модульные с мок объектом
- Интеграционные

Тесты:
- конвертация объекта в json
- работа контроллера
- работа сервиса

Что тестируем
- Пользователь. Создание. Ошибка уникальности логина
- Пользователь. Создание. Успех. Возвращаем нового пользователя
- Пост. Получение страницы постов. Успех
- Пост. Получение данных поста со средней оценкой.
- Пост. Удаление
